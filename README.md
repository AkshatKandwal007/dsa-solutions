# DSA Solutions

A collection of my data structures and algorithms solutions, organized by
problem-solving pattern. Solved in C, following a pattern-first approach.

## Method
For each problem I work through a consistent loop:
1. Identify the underlying pattern
2. Establish a brute-force baseline
3. Attempt an optimized solution under time constraints
4. Study the optimal solution and note the key insight
5. Re-solve later for retention

## Structure
Solutions are grouped by pattern (two pointers, sliding window, binary
search, dynamic programming, and so on). Each file corresponds to a single
problem.

## Progress
Updated regularly as I work through problems.
